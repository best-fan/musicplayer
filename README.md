# musicplayer
本地音乐播放器

## 项目介绍
基于C#的本地播放器WinForm项目功能包含 卡拉OK字母 随机播放、单曲循环、顺序播放、列表循环播放模式

* 天气数据:彩云天气
* 定位数据:高德地图
## 项目技术
* MVP架构
* OKhttp  
* Fastson
* AndPermission
* 自定义View
